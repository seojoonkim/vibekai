# 🏆 VibeDojo 프로젝트 쇼케이스

VibeDojo에서 만든 프로젝트를 공유하고 피드백을 받아보세요!

## 📋 제출 가이드라인

### 필수 요소

1. **프로젝트 README**
   - 프로젝트 이름과 한줄 설명
   - 무엇을 만들었는지 (What)
   - 왜 만들었는지 (Why)
   - 어떻게 만들었는지 (How)

2. **스크린샷/데모**
   - 최소 1장의 스크린샷
   - 또는 데모 GIF/영상

3. **라이브 URL** (권장)
   - Vercel, Netlify, GitHub Pages 등

### 제출 방법

#### Option 1: PR로 제출
```bash
# 1. Fork & Clone
git clone https://github.com/YOUR_USERNAME/vibedojo-dev.git

# 2. 폴더 생성
mkdir -p showcase/projects/YOUR_USERNAME/PROJECT_NAME

# 3. README.md 작성 (템플릿 참고)
cp showcase/TEMPLATE.md showcase/projects/YOUR_USERNAME/PROJECT_NAME/README.md

# 4. PR 생성
```

#### Option 2: Discord 공유
`#project-showcase` 채널에 템플릿 형식으로 공유
