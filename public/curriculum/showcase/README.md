## 📋 제출 가이드

### 요구사항

| 항목 | 필수 | 설명 |
|-----|-----|------|
| GitHub 레포 | ✅ | 공개 레포지토리 |
| README.md | ✅ | 프로젝트 설명, 스크린샷 포함 |
| 배포 URL | 권장 | Vercel, Netlify 등 |
| 학습 회고 | 권장 | 어떤 챕터가 도움됐는지 |

### 제출 방법

1. 아래 템플릿으로 프로젝트 폴더 생성
2. 이 폴더에 PR 제출
3. 리뷰 후 머지!
