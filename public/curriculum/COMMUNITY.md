## 🏆 프로젝트 쇼케이스

커리큘럼을 완료하고 만든 프로젝트를 공유하세요!

### 제출 가이드라인

1. **GitHub 레포** 필수
2. **README.md** 포함 (프로젝트 설명, 스크린샷)
3. **배포 URL** (선택, 권장)
4. **학습 회고** (어떤 챕터가 도움됐는지)

### 쇼케이스 제출

```markdown
## 프로젝트명: [프로젝트 이름]

### 설명
[한 줄 설명]

### 사용 기술
- Claude Code
- [기타 기술들]

### 링크
- GitHub: [URL]
- 배포: [URL]

### 학습 회고
가장 도움된 챕터: Chapter XX
챌린지: [어려웠던 점]
해결 방법: [어떻게 극복했는지]
```

Discord #showcase 채널 또는 [showcase/](./showcase/) 폴더에 PR로 제출!
